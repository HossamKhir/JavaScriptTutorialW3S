Remainder


        var x = 5;
        var y = 2;
        var z = x % y;
        var z = x + y;  // addition
        var z = x - y;  // subtraction
        var z = x * y;  // multiplication
        var z = x * y;  // division

Incrementing

The increment operator (++) increments numbers 


            var x = 5;
            x++;    // postfix, return then increment
            var z = x;
            ++x;    // prefix, increment then return

Decrementing

The decrement operator (--) decrements numbers 


            var x = 5;
            x--;    // postfix, return then decrement
            var z = x;
            --x;    // prefix, decrement then return

Exponentiation

The exponentiation operator (**) raises the first operand to the power of the second operand 


            var x = 5;
            var z = x ** 2; // result is 25
            // same result as
            var z = Math.pow(x,2);  // result is 25
