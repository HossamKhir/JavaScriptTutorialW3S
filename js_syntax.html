Values can be either literals 'fixed' or 'variable'

Literals

Numbers: could be floating point or not:
10.50
1001

Strings: could be written within single or double quotes
'string'
"string"

Variables

JavaScript uses the var keyword to declare variables 

        
            var x;
            x = 6;  // = is called an assignment operator
            var y = 5;

JavaScript uses arithmetic operators ( + - * / ) to compute values

Expressions

An expression is a combination of values, variables, and operators, which computes to a value 

        
            5 * 10; // this is an expression
            x * 15; // this is also an expression
            "John" + " " + "Doe";   // this one too, and it evaluates to "John Doe"

Keywords

JavaScript keywords are used to identify actions to be performed

Comments

Code after double slashes // or between /* and */ is treated as a comment

Identifiers

In JavaScript, identifiers are used to name variables (and keywords, and functions, and labels)
All JavaScript identifiers are case sensitive.
